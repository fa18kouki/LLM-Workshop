# AIエージェント構築研修

このリポジトリは、「Pythonによる実践的AIエージェント開発」研修の教材です。

## 概要

本研修では、4回のセッションを通じて、Pythonを使った簡易的なAIエージェントの構築を行います。
生成AIの基本から、LLM API連携、外部ツール連携（MCP）まで、実践的なスキルをハンズオン形式で習得します。

## ディレクトリ構成

- `.devcontainer/`: 開発環境を統一するためのDevContainer設定
- `session-01/`: 第1回：環境構築と基礎理解
- `session-02/`: 第2回：LLM API連携と対話機能の実装
- `session-03/`: 第3回：MCP実装による機能拡張
- `session-04/`: 第4回：高度な機能実装と総括
- `docs/`: 研修資料（スライドなど）
- `assets/`: 共通のアセット（画像など）

## 開発環境のセットアップ

1.  Docker Desktopをインストールします。
2.  Visual Studio Codeをインストールします。
3.  VSCode拡張機能「Dev Containers」をインストールします。
4.  このリポジトリをクローンし、VSCodeで開きます。
5.  コマンドパレット（`Ctrl+Shift+P` または `Cmd+Shift+P`）を開き、「Reopen in Container」を実行します。

これにより、コンテナ内に開発環境が自動的に構築されます。
